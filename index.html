Projects:

  • GA
    Applying genetic programming to optimization problems.
    Currently contains a standard genetic algorithm implementation for the MAXSAT problem.
  • QLearning
    Reinforcement learning agent learning to navigate a maze containing rewards and punishments using the Q-learning technique.
    Based on the sections about Q-learning in the Reinforcement Learning chapter of Artificial Intelligence: A Modern Approach.
  • dota2-deathmap
    Interactive D3 visualizations for a generated DotA 2 death map.
    Visualizes player deaths using a game map overlaid by a generated scatter plot, and shows the distribution of deaths across player tiers for a selected area on the map using a dynamically updated pie chart.
  • PhiVis
    Interactive D3 visualizations for the PHI-base data set.
    Given a selection of diseases, visualizes the number of PHI-base entries per disease, the number of entries per gene per disease, and the common phenotypical effects between these diseases.
